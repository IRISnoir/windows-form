# MyWordPad

Ứng dụng soạn thảo văn bản đơn giản được xây dựng bằng **C# Windows Forms** (.NET Framework 4.7.2).

## Yêu cầu

- Windows OS
- Visual Studio 2022+ hoặc .NET Framework 4.7.2

## Nguồn lấy ảnh
[lucide.dev](https://lucide.dev/icons/)

- Color: #000000
- Stroke width: 2px
- Size: 256px

> Lưu ý: Ảnh mặc định là PNG, cần chuyển sang ICO nếu muốn sử dụng.

## Cách chạy

1. Mở `MyWordPad.slnx` bằng Visual Studio
2. Nhấn `F5` để build và chạy
